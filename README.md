# Business web app for Artyom

## Set up and Get started

### Set up

- Clone the repository.

```bash
    git clone https://github.com/ArtyomAch/business-app.git
```

- Install the dependencies.

```bash
    npm install
    #or
    yarn
```

### Get started

- Run the project

```bash
    npm start
    #or
    yarn start
```

- Build the project

```bash
    npm run build
    #or
    yarn build
```
